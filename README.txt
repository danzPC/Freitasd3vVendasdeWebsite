COMO ATUALIZAR OS EXEMPLOS (DEMONSTRAÇÃO)

1) Link do botão (vai direto para o Vercel)
- No index.html, procure por: href="https://seu-link-landing.vercel.app/"
- Troque pelo seu link real.
Exemplo (Advogados):
https://advogado-exemplo-freitas-d3v.vercel.app/

2) Imagem do card (thumbnail)
- Coloque um print em: assets/previews/
- Troque o src do <img class="serviceThumb" ...> no index.html.
Recomendado: .webp (leve), ou .png/.jpg.
